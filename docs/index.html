Microbiome Digest

A daily automated digest on new microbiome papers, specifically focusing on host-microbiome interactions, functions of the microbiome, associations with specific disorders and microbiome engineering, topics of interest of the Lloréns-Rico lab. The pipeline runs every morning, ranks new papers from multiple sources, and narrates them into a roughly hour-long episode.

ⓘ

Built on free resources only, so the audio is best used for triage: find papers worth reading, then read the originals.

Domain not in any RSS feed — pipeline can’t see it. Add to rss_sources or check extra_rss_sources.json for auto-discovered feeds.
low ranking
In RSS but cut below the episode cap. Add keywords to absolute_title_keywords or increase max_items_total.

Workflow hasn’t run yet — diagnosis appears within ~2 minutes.
